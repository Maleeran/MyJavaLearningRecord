#### 2.基本数据类型（Primitive Types）

| 类型        | 占用内存 | 范围              | 示例                                  |
| --------- | ---- | --------------- | ----------------------------------- |
| `byte`    | 1字节  | -128\~127       | `byte b = 127;`                     |
| `short`   | 2字节  | -32,768\~32,767 | `short s = 32000;`                  |
| `int`     | 4字节  | -2³¹\~2³¹-1     | `int age = 18;`                     |
| `long`    | 8字节  | 非常大             | `long population = 7_000_000_000L;` |
| `float`   | 4字节  | 小数              | `float f = 3.14f;`                  |
| `double`  | 8字节  | 精度更高            | `double d = 2.71828;`               |
| `char`    | 2字节  | 单个字符            | `char ch = 'A';`                    |
| `boolean` | 1字节  | true/false      | `boolean isPassed = true;`          |

---

#### ✅ 今日重点
- Java有8种基本数据类型，整型默认是 int，浮点型默认是 double
- 自动类型转换：byte → short → int → long → float → double
- 强制转换需要加 `(目标类型)`，如 `(int) 3.14`

#### ✅ 易错点
- float 要加 `f`：`float f = 3.14f;`
- long 要加 `L`：`long l = 123456789L;`
- char 是单引号：`'A'`，不是双引号
